无限计算的错觉：市场回调

普遍的说法认为，计算能力的丰富程度将无限增长，这种观点基于将摩尔定律线性推断应用于复杂的神经架构，从而将估值推向平流层。然而，在下一代张量处理单元所需的稀土元素供应链中隐藏着一个系统性漏洞。

对比：虽然软件将复杂性抽象为优雅的界面，但硬件的物理现实受到地质和地缘政治的残酷限制。市场正在为软件利润率定价，而忽略了硬件现实。

"我们运作在一种共识幻觉中，即硅被视为可以无限复制的软件。"

当不可避免的瓶颈出现时，现有的、高效的模型的溢价将飙升。对没有必要基础设施而被淘汰的恐惧，将推动资本从投机性的软件包装器大规模重新分配到硬资产 IP。

  • 关键组件供应链的脆弱性。
  • 过度依赖集中式、高能耗的数据中心。
  • 即将出台的关于功耗的监管限制。

在硬件优化算法和本地化、低功耗推理模型方面的战略性 IP 收购，将把有韧性的企业与那些暴露在即将到来的调整中的企业区分开来。
